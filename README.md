# 🛡️ Modern Defense: Operations (现代防御：行动)

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![Pygame](https://img.shields.io/badge/Pygame-2.x-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

这是一款基于 **Python** 和 **Pygame** 引擎独立开发的 2D 现代战争题材策略塔防游戏。本项目不仅复刻了经典塔防的核心布局与资源运营玩法，还创新性地融入了 **Roguelike（肉鸽）局内增益机制**、**局外科技树天赋养成** 以及 **硬核的战术微操系统**。

## ✨ 核心游戏特色

* 🗺️ **多维战役关卡**：内置 7 大硬核关卡，涵盖从线性阻击、多路包夹到无视地形的空中突袭与迷宫防线。
* ♾️ **无尽深渊 (Endless Roguelike)**：在第 7 关中引入了 Roguelike 机制，每存活 5 波将触发“战术空投”，玩家需在随机的 3 种全局 Buff 中进行抉择。
* 🎯 **智能火控微操**：拒绝被动挂机！玩家可随时选中防御塔，按 `[T]` 键在 `FIRST(打最前)`、`STRONG(打最强)`、`CLOSE(打最近)` 三种火控逻辑中即时热切换。
* 🚀 **轨道卫星打击**：危机时刻可呼叫战术全屏大招，附带真实的屏幕震动特效与多层数学演算的粒子爆炸渲染，造成毁灭性真实伤害。
* 💰 **高风险经济博弈**：独特的 `Early Call（提前呼叫）` 机制，在波次空档期强行按下回车呼叫敌军，可根据节省的时间按比例转化为高额战术资金。
* 🧬 **科技实验室 (Tech Lab)**：局外长线养成系统。利用通关获取的科技点永久强化弹药伤害、工程造价与基地装甲，支持自由洗点。

## 🧱 战术单位情报

### 🏗️ 防御建筑 (Towers)
1. **机枪巢 (MG Nest)**：高射速，专精于清理大规模脆弱虫群。
2. **狙击塔 (Sniper)**：超远射程与极致单发爆发，点杀精英机甲的绝对主力。
3. **激光塔 (Laser)**：持续锁定光束，快速融化重型装甲。
4. **诱饵路障 (Barricade)**：全场唯一可放置在道路上的特殊单位。无攻击力但血量极厚，用于物理卡位与拉扯仇恨。

### 👾 敌对势力 (Hostiles)
* **侦察车 (Buggy) / 自爆卡车 (Kamikaze)**：高速移动，后者一旦靠近将引发自毁范围爆炸。
* **攻城机甲 (Mech)**：高额血量，可在远距离炮击大本营。
* **飞行无人机 (Swarm Drone)**：无视常规道路网络，沿绝对直线越过地形突袭基地。
* **护盾发生器 (Aegis Generator)**：为大范围敌军提供免伤护盾（代码内置同类排斥逻辑，杜绝护盾死锁）。
